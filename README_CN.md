- # loseq-rss-reader
  - 在 Logseq 中阅读 RSS 内容并导入至 Logseq
- ## 功能
  - 添加 RSS 订阅源
  - 删除 RSS 订阅源
  - 文章列表
  - 文章阅读
  - 页面主题（支持浅色主题和深色主题）
  - 保存文章至 Logseq
